# Practicum_DA_projects
Practicum Data Analyst Projects

**1) Анализ и оптимизация марект плейса (Project_Marketplace.ipynb)**
- Задачи:
  - Анализ загруженных данных (EDA)
  - Комплексный анализ метрик и юнит-экономики
  - Выявление драйверов роста и узких мест в бизнесе
  - Определение приоритетных направлений развития
  - Внедрение data-driven подхода в принятие решений

- Результаты:
  - Выявлено снижение прибыли маркетплейса через анализ воронки продаж
  - Определен проблемный канал привлечения пользователей, влияющий на выручку
  - Проведено A/B-тестирование гипотез по увеличению прибыли канала
  - Подтверждена статистическая значимость изменений по ключевым метрикам
  - Разработаны рекомендации по масштабированию успешных решений
 
- Инструменты: Python, Jupyter Notebook
- Библиотеки: pandas, numpy, matplotlib
- Статистический анализ: scipy, statsmodels
- Методологии: воронки событий, A/B-тестирование

**2) Анализ эффективности образовательного контента для платформы оглайн-торговли финансовыми активами (Project_FinTech.ipynb)**
- Задачи:
  - Проверка гипотезы о влиянии дополнительного обучения на прибыльность
  - Анализ пользовательского поведения
  - Оценка эффективности образовательных сценариев
 
- Результаты:
  - Выявлены ключевые точки оттока пользователей 
  - Проведен A/B-тест по гипотезе об увеличении прибыли через дополнительное обучение
  - Проанализированы ключевые метрики (средний депозит на пользователя)

- Инструменты: Python, Jupyter Notebook
- Библиотеки: pandas, numpy, matplotlib
- Статистический анализ: scipy, statsmodels
- Методологии: воронки событий, A/B-тестирование
